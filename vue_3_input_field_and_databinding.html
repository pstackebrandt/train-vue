Vue 3 Two-Way Data Binding

Purpose

Learn how to create bidirectional data binding between form inputs and Vue data variables.

Content

This example demonstrates: v-model directive, data() function, reactive variables, and real-time input synchronization.

We get the variables from the app: x = {{ x }} and y = {{ y }}.

Calculate: x * y = {{ x * y }}

After changing the input fields, the variables and their representation on the page will be updated.
