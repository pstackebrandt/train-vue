🔍 How v-model Two-Way Binding Works

  1. Create Connection: v-model="x" creates bidirectional link between input field and data variable
  2. Direction 1 - Input → Data: User types "99" → x becomes 99 instantly
  3. Direction 2 - Code → Input: Button sets x = 50 → input field shows "50" instantly
  4. Automatic Updates: Both directions trigger reactive updates to all expressions using the variable
💡 v-model vs {{ }}:
• v-model="x" = two-way binding
• {{ x }} = one-way display only
🔄 Key Insight:
v-model combines :value and @input into one simple directive for perfect synchronization.
