Use conditional evaluation

We can use conditional evaluation to check if the variables are set and whether they are numbers.
Thereby we can deactivate elements or show error messages. (We get the variables from the app: x = {{ x }} and y = {{ y }}.)

Bitte füllen Sie alle Felder aus!

Bitte nur mit Zahlen arbeiten!

{{ x * y }}

Calculate: x * y = {{ x * y }}

After changing the input fields, the variables and their representation on the page will be updated.
